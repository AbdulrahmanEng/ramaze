Ramaze is working correctly with this application, now you can start working

To start you can modify:

  1. template/index.xhtml, which this text
  2. src/elements/page.rb, which is the Element around this text
  3. src/controllers/main.rb, where you can change the header of this page and the notemplate action
  4. src/model.rb, which is the the persistent backend for your app

You can also read the tutorial in the doc/ directory of ramaze or browse the rdocs
